# 한자 암기 비법: 뇌과학을 활용한 공부법

한자 공부의 가장 큰 적은 '망각'입니다. 외워도 외워도 자꾸 까먹는다면, 무작정 쓰기만 하는 방식에서 벗어나 뇌과학적인 접근을 시도해 보세요.

## 1. 파자(破字)로 스토리 만들기
한자를 쪼개서 이야기를 만드세요.
예를 들어 **'쉴 휴(休)'**는 '사람(人)'이 '나무(木)'에 기대어 쉬고 있는 모습입니다. 이렇게 이미지를 떠올리며 스토리를 입히면 뇌는 정보를 훨씬 더 잘 저장합니다.

## 2. 분산 반복 학습 (Spaced Repetition)
오늘 100번 쓰는 것보다, 오늘 10번, 내일 5번, 일주일 뒤 3번 보는 것이 훨씬 효과적입니다. 뇌는 잊어버릴 만할 때 다시 자극을 주면 기억을 장기 기억으로 전환합니다.

## 3. 맥락 속에서 익히기
낱글자만 외우지 말고 단어와 문장 속에서 익히세요. '학교', '학생', '학습' 처럼 '배울 학(學)'이 들어가는 단어를 묶어서 공부하면 연결 고리가 생겨 인출이 쉬워집니다.

## 결론
암기는 머리가 나빠서 못하는 것이 아니라, 방법이 틀려서 힘든 것입니다. 똑똑하게 공부해서 한자 마스터가 되어보세요!

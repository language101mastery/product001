# 천자문 학습 애플리케이션 블루프린트

## 개요

이 문서는 천자문 학습 웹 애플리케이션의 설계, 기능, 개발 계획을 정의합니다. 사용자는 이 애플리케이션을 통해 한국 한자, 중국 간체자, 일본 신자체를 학습하고 관련 블로그 글을 읽을 수 있습니다.

## 1. 초기 버전 (v1.0)

### 1.1. 디자인 및 스타일

- **테마:** 라이트 모드와 다크 모드를 지원하며, 사용자의 시스템 설정을 기본값으로 따릅니다.
- **폰트:** 한자 학습의 가독성을 높이기 위해 Noto Sans CJK 글꼴을 사용합니다.
- **레이아웃:** 중앙 정렬된 컨테이너를 사용하여 콘텐츠를 표시하며, 모바일 반응형 디자인을 적용합니다.
- **시각적 요소:** 카드 UI, 미묘한 그림자 효과, 인터랙티브한 버튼으로 현대적인 미학을 추구합니다.

### 1.2. 핵심 기능

- **천자문 학습 카드:**
  - 중앙에 천자문 구절과 뜻을 표시하는 웹 컴포넌트(`cheonjamun-card`)를 사용합니다.
  - 이전/다음 버튼으로 250개의 천자문 구절을 탐색할 수 있습니다.
- **문자셋 선택:**
  - 한국 한자(Hanja), 중국 간체자(ChineseS), 일본 신자체(Japanese) 중에서 선택하여 학습할 수 있습니다.
  - 선택된 문자셋에 따라 해당 글자와 폰트가 동적으로 변경됩니다.
- **상세 정보 표시:**
  - 현재 표시된 천자문 구절에 포함된 각 한자의 음과 뜻을 그리드 형태로 보여줍니다.
- **블로그:**
  - 한자 학습과 관련된 정보를 제공하는 블로그 페이지가 있습니다.
  - Markdown으로 작성된 게시물을 동적으로 불러와 렌더링합니다.

## 2. 홈페이지 및 내비게이션 개편 (v1.1)

**요청 사항:** 홈페이지 구조를 변경하여 `index.html`은 환영 메시지만 표시하고, 기존 천자문 학습 기능은 별도의 페이지로 분리합니다. 내비게이션 메뉴를 "Home", "1000", "blog"로 개편합니다.

## 3. 홈페이지 디자인 개선 (v1.2)

**요청 사항:** `index.html`의 환영 문구를 `1000.html`의 천자문 카드와 유사한 스타일로 변경하고, 페이지 제목을 "천자문 공부"로 설정합니다.

---

## 4. 다국어 지원 (Internationalization) (v1.3) - 현재 계획

**요청 사항:** 웹사이트를 한국어, 영어, 중국어, 일본어로 제공합니다.

### 4.1. 작업 단계

1.  **언어 데이터 파일 생성:**
    -   `locales` 폴더를 생성합니다.
    -   내부에 `ko.json`, `en.json`, `zh.json`, `ja.json` 파일을 만들고, UI 텍스트(내비게이션, 제목, 환영 메시지 등)에 대한 번역을 키-값 형태로 저장합니다.

2.  **HTML 구조 수정:**
    -   번역이 필요한 모든 텍스트 요소를 식별할 수 있도록 `data-i18n-key` 속성을 추가합니다.
    -   사용자가 언어를 선택할 수 있는 UI(버튼 그룹)를 내비게이션 영역에 추가합니다.

3.  **`main.js` 로직 구현:**
    -   페이지 로드 시 사용자의 `localStorage`에 저장된 언어 설정이나 브라우저의 기본 언어 설정을 확인하여 기본 언어를 설정하는 기능을 추가합니다.
    -   선택된 언어에 맞는 `.json` 파일을 비동기적으로 불러오는 함수를 구현합니다.
    -   불러온 번역 데이터를 기반으로 `data-i18n-key` 속성을 가진 모든 요소의 텍스트를 동적으로 교체하는 함수를 작성합니다.
    -   언어 선택 버튼에 이벤트 리스너를 추가하여, 버튼 클릭 시 언어를 변경하고 `localStorage`에 선택된 언어를 저장하도록 합니다.

4.  **`style.css` 스타일 추가:**
    -   새로 추가된 언어 선택 UI를 기존 디자인과 어울리도록 스타일을 적용합니다.

5.  **GitHub 리포지토리 업데이트:**
    -   모든 변경 사항을 스테이징하고 "Implement internationalization (i18n)" 커밋 메시지와 함께 원격 리포지토리에 푸시합니다.

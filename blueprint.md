# 천자문 학습 애플리케이션 블루프린트

## 개요

이 문서는 천자문 학습 웹 애플리케이션의 설계, 기능, 개발 계획을 정의합니다. 사용자는 이 애플리케이션을 통해 한국 한자, 중국 간체자, 일본 신자체를 학습하고 관련 블로그 글을 읽을 수 있습니다.

## 1. 초기 버전 (v1.0)

### 1.1. 디자인 및 스타일

- **테마:** 라이트 모드와 다크 모드를 지원하며, 사용자의 시스템 설정을 기본값으로 따릅니다.
- **폰트:** 한자 학습의 가독성을 높이기 위해 Noto Sans CJK 글꼴을 사용합니다.
- **레이아웃:** 중앙 정렬된 컨테이너를 사용하여 콘텐츠를 표시하며, 모바일 반응형 디자인을 적용합니다.
- **시각적 요소:** 카드 UI, 미묘한 그림자 효과, 인터랙티브한 버튼으로 현대적인 미학을 추구합니다.

### 1.2. 핵심 기능

- **천자문 학습 카드:**
  - 중앙에 천자문 구절과 뜻을 표시하는 웹 컴포넌트(`cheonjamun-card`)를 사용합니다.
  - 이전/다음 버튼으로 250개의 천자문 구절을 탐색할 수 있습니다.
- **문자셋 선택:**
  - 한국 한자(Hanja), 중국 간체자(ChineseS), 일본 신자체(Japanese) 중에서 선택하여 학습할 수 있습니다.
  - 선택된 문자셋에 따라 해당 글자와 폰트가 동적으로 변경됩니다.
- **상세 정보 표시:**
  - 현재 표시된 천자문 구절에 포함된 각 한자의 음과 뜻을 그리드 형태로 보여줍니다.
- **블로그:**
  - 한자 학습과 관련된 정보를 제공하는 블로그 페이지가 있습니다.
  - Markdown으로 작성된 게시물을 동적으로 불러와 렌더링합니다.

## 2. 홈페이지 및 내비게이션 개편 (v1.1)

**요청 사항:** 홈페이지 구조를 변경하여 `index.html`은 환영 메시지만 표시하고, 기존 천자문 학습 기능은 별도의 페이지로 분리합니다. 내비게이션 메뉴를 "Home", "1000", "blog"로 개편합니다.

### 2.1. 작업 단계

1.  **`index.html` 파일 분리:**
    -   기존 `index.html`의 내용을 `1000.html`이라는 새 파일로 복사합니다.
    -   `index.html`은 환영 문구와 새로운 내비게이션 메뉴만 포함하도록 수정합니다.

2.  **내비게이션 메뉴 개편:**
    -   `index.html`, `1000.html`, `blog.html`, `post.html` 파일에 포함된 내비게이션 메뉴를 일관되게 수정합니다.
    -   새로운 메뉴는 **Home** (`index.html`), **1000** (`1000.html`), **blog** (`blog.html`) 세 가지 링크로 구성됩니다.

## 3. 홈페이지 디자인 개선 (v1.2) - 현재 계획

**요청 사항:** `index.html`의 환영 문구를 `1000.html`의 천자문 카드와 유사한 스타일로 변경하고, 페이지 제목을 "천자문 공부"로 설정합니다.

### 3.1. 작업 단계

1.  **`index.html` 구조 변경:**
    -   `<h1>` 태그의 내용을 "천자문 공부"로 수정합니다.
    -   기존 환영 문구를 `<div>`로 감싸고, `welcome-card` 클래스를 추가하여 카드 스타일을 적용할 수 있도록 합니다.
    -   새로운 카드 내부에 환영 문구를 `<h2>` 태그로 표시합니다: "천자문 공부 페이지에 오신 것을 환영합니다."

2.  **`style.css` 스타일 추가:**
    -   `welcome-card` 클래스를 위한 CSS 스타일을 추가합니다.
    -   `cheonjamun-card`의 스타일(배경색, 그림자, 패딩, 테두리 등)을 참조하여 일관된 디자인을 유지합니다.

3.  **GitHub 리포지토리 업데이트:**
    -   모든 변경 사항을 스테이징하고 "Style index page welcome message as a card" 커밋 메시지와 함께 원격 리포지토리에 푸시합니다.
